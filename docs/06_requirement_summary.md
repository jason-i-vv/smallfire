# 星火量化系统 - 项目需求总览

## 1. 项目概述

**项目名称：** 星火量化
**项目目标：** 辅助交易的微型量化交易系统，星星之火，点燃你的交易人生。

## 3. 模块详细需求

### 3.1 行情抓取模块

| 需求项 | 描述 | 优先级 |
|--------|------|--------|
| bybit市场 | 获取币圈合约行情，周期15m/1h | P0 |
| A股市场 | 获取A股日线行情 | P1 |
| 美股市场 | 获取美股日线行情 | P1 |
| 数据缓存 | 本地存储K线数据，减少API调用 | P0 |
| 热度管理 | 按热度筛选100-200个标的 | P1 |
| 工厂模式 | 管理多个行情抓取器 | P0 |

### 3.2 策略分析模块

| 需求项 | 描述 | 优先级 |
|--------|------|--------|
| 箱体突破策略 | 识别交易箱体，监测突破信号 | P0 |
| 趋势策略 | EMA均线判断多空趋势 | P0 |
| 阻力支撑策略 | 识别关键价位，监测突破 | P1 |
| 量价异常策略 | 监测价格/成交量异常波动 | P1 |
| 工厂模式 | 管理多个策略分析器 | P0 |

### 3.3 交易跟踪模块

| 需求项 | 描述 | 优先级 |
|--------|------|--------|
| 开仓管理 | 根据信号创建交易跟踪 | P0 |
| 止损策略 | ATR止损 + 固定止损 | P0 |
| 止盈策略 | 固定止盈 + 移动止损 | P0 |
| 仓位管理 | 固定比例/凯利公式 | P1 |
| 风控检查 | 每日交易次数/最大持仓/回撤限制 | P0 |
| 统计分析 | 胜率/盈亏比/最大回撤等 | P0 |

### 3.4 实时监测模块

| 需求项 | 描述 | 优先级 |
|--------|------|--------|
| 订阅管理 | 策略订阅/退订价格监测 | P0 |
| 事件触发 | 价格达到目标时触发事件 | P0 |
| 订阅计数 | 多个策略共享监测器 | P0 |
| 服务重启 | 恢复活跃监测器 | P1 |

### 3.5 通知模块

| 需求项 | 描述 | 优先级 |
|--------|------|--------|
| 飞书通知 | 发送交易信号到飞书群 | P0 |
| 信号推送 | 实时推送新信号 | P0 |
| 汇总推送 | 定时汇总推送 | P1 |

### 3.6 前端控制台

| 需求项 | 描述 | 优先级 |
|--------|------|--------|
| 登录认证 | 用户登录/Token认证 | P0 |
| 信号展示 | 信号列表/详情/筛选 | P0 |
| 持仓监控 | 实时持仓/盈亏展示 | P0 |
| 交易统计 | 权益曲线/胜率统计 | P0 |
| K线图表 | 实时K线/箱体标注 | P0 |
| 策略配置 | 策略开关/参数调整 | P1 |

## 4. 技术指标要求

### 4.1 性能指标

| 指标 | 要求 |
|------|------|
| K线更新延迟 | < 1秒 |
| 信号生成延迟 | < 5秒 |
| API响应时间 | < 200ms |
| 同时监测标的 | 500+ |
| 数据存储 | 支持3个月K线数据 |

### 4.2 可用性指标

| 指标 | 要求 |
|------|------|
| 系统可用性 | 99.5% |
| 服务重启恢复 | < 30秒 |
| 数据备份 | 每日自动备份 |
