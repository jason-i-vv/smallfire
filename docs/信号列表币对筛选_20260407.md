# 信号列表币对筛选功能

## 变更日期
2026-04-07

## 需求描述
信号列表页面 `/signals` 支持按币对（标的）进行筛选。用户在选定市场后，可以通过下拉框选择特定币对，快速过滤该币对下的所有信号。

## 变更内容

### 后端
1. **SignalQuery 模型** (`internal/models/signal.go`)
   - 新增 `SymbolCode string` 字段，用于按标的代码筛选

2. **SignalHandler** (`internal/handler/signal_handler.go`)
   - `GetSignals` 方法新增解析 `symbolCode` query 参数

3. **SignalRepo** (`internal/repository/signal_repo.go`)
   - `Query` 方法新增 `sy.symbol_code = $N` WHERE 条件

### 前端
4. **SignalList 页面** (`starfire-frontend/src/views/signals/SignalList.vue`)
   - 筛选栏新增可搜索、可清空的币对下拉框
   - 切换市场时自动加载该市场的标的列表（调用 `symbolApi.listByMarket`）
   - 切换市场时自动重置已选币对
   - 页面加载时根据默认市场自动加载标的列表

## API 变更
- `GET /api/v1/signals` 新增可选参数 `symbolCode`：按标的代码筛选信号
